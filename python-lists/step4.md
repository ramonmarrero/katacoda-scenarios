
The built-in function len() also applies to lists:

`letters = ['a', 'b', 'c', 'd']`{{execute}} 

`len(letters)`{{execute}} 

It is possible to nest lists (create lists containing other lists), for example:

`a = ['a', 'b', 'c']`{{execute}} 

`n = [1, 2, 3]`{{execute}} 

`x = [a, n]`{{execute}} 

`x`{{execute}} 

`x[0]`{{execute}} 

`x[0][1]`{{execute}}
